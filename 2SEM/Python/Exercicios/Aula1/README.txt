1) Contagem Regressiva: Crie um algoritmo que solicite um número e
imprima os valores em ordem decrescente.
2) Adivinhe o Número: Crie um jogo onde o programa gera um número
aleatório e o jogador deve adivinhar o número. Utilize a biblioteca
random.
3) Tabuada: Peça ao usuário um número e, em seguida, use um loop para
exibir a tabuada desse número de 1 a 10.
4) Sequência de Fibonacci: Crie um programa que solicite um número ao
usuário e, em seguida, use um loop while para gerar e exibir a
sequência de Fibonacci até o número especificado.
5) Fatorial: Peça ao usuário um número e calcule o fatorial desse número
usando um loop while.
