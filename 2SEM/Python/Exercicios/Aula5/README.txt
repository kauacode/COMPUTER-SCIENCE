Exercício 1: Crie uma classe chamada Retangulo com métodos para calcular a área e o
perímetro do retângulo. A classe deve ser inicializada com os valores dos lados do
retângulo.
Exercício 2: Crie uma classe chamada Circulo com métodos para calcular a área e o
perímetro do círculo. A classe deve ser inicializada com o raio do círculo.
Exercício 3: Crie uma classe chamada ContaBancaria com métodos para depositar,
sacar e verificar o saldo da conta.
Exercício 4: Crie uma classe chamada Pessoa com métodos para definir o nome e a
idade da pessoa. Em seguida, crie um objeto pessoa e exiba seu nome e idade.
Exercício 5: Crie uma classe chamada Carro com métodos para definir a marca, o
modelo e o ano do carro. Em seguida, crie um objeto carro e exiba suas informações.
Exercício 6: Crie uma classe chamada Triangulo com métodos para calcular a área e o
perímetro do triângulo. A classe deve ser inicializada com os valores dos lados do
triângulo.
Exercício 7: Crie uma classe chamada Livro com métodos para definir o título, o autor e
o ano de publicação do livro. Em seguida, crie um objeto livro e exiba suas
informações.
Exercício 8: Crie uma classe chamada ContaCorrente que herda da classe
ContaBancaria (do exercício 3) e inclua um método para verificar se o cliente possui
um limite de crédito. Se o cliente possuir um limite de crédito, permita que ele faça
saques mesmo que o saldo seja insuficiente, descontando do limite de crédito.
